# Beitrag zu *Mein-Aquarium* 🐟🌿

Vielen Dank, dass du darüber nachdenkst, zu *Mein-Aquarium* beizutragen!  
Dieses Projekt lebt davon, dass die Community Ideen einbringt, Fehler findet und neue Features entwickelt.  

---

## 🚀 Wie du beitragen kannst
- **Fehler melden (Issues):**  
  - Nutze GitHub-Issues, um Bugs zu melden oder Feature-Ideen vorzuschlagen.  
  - Beschreibe so genau wie möglich, wie man den Fehler reproduzieren kann.  

- **Code beitragen (Pull Requests):**  
  1. Forke das Repository.  
  2. Erstelle einen Feature-Branch (`git checkout -b feature/dein-feature`).  
  3. Committe deine Änderungen mit klaren, beschreibenden Nachrichten.  
  4. Erstelle einen Pull Request.  

- **Dokumentation verbessern:**  
  Auch kleine Korrekturen an der README, bessere Erklärungen oder Screenshots sind willkommen.  

---

## 🛠 Entwicklungsumgebung
- **Sprache:** Swift 5.x  
- **IDE:** Xcode 16 (oder neuer)  
- **Plattformen:** macOS (SwiftUI), iOS (SwiftUI), später auch Android  

---

## 📐 Konventionen
- **Code-Style:** Halte dich an gängige Swift-Konventionen (SwiftLint, falls aktiviert).  
- **Commits:** Kurze, prägnante Nachrichten auf Englisch oder Deutsch.  
- **Branches:**  
  - `main` = stabile Version  
  - `feature/...` = neue Features  
  - `fix/...` = Bugfixes  

---

## 🤝 Zusammenarbeit
- Bitte halte dich an den [Verhaltenskodex](CODE_OF_CONDUCT.md).  
- Diskussionen, Vorschläge und Feedback sind ausdrücklich erwünscht.  
- Entscheidungen der Maintainer werden transparent kommuniziert.
