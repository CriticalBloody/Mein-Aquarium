# Mein Aquarium

**Mein Aquarium** ist eine moderne iOS-App zur Verwaltung und Überwachung deiner Aquarien, inklusive Wasserwerte, Bewohner und Pflanzen.

## Features

- 📊 Übersichtliche Anzeige deiner Wasserwerte als Chart & Liste
- 🐟 Verwaltung von mehreren Aquarien und deren Bewohnern
- ✏️ Einfache Eingabe und Bearbeitung von Wassertests
- ⚠️ Warnungen, wenn Werte außerhalb der Sollbereiche liegen
- 📝 Notizenfunktion pro Testmessung
- 🎛️ Anpassbare Sollwertbereiche (global & pro Aquarium, demnächst)

## Installation

1. **Xcode öffnen:**  
   Projekt in Xcode 15 (oder neuer) öffnen.
2. **Abhängigkeiten:**  
   Keine externen Abhängigkeiten erforderlich.
3. **Simulator:**  
   Auf iOS-Simulator oder echtem Gerät (iOS 17+) starten.
